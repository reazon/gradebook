Uploading/Importing scores to the Gradebook in bSpace

In the Gradebook, both students and instructors can view current scores.

Note: To complete the procedures in this document, you need to be assigned a role that has the necessary permissions. For a basic understanding of roles and permissions, see the Permissions and Roles section of Help.

From within the Gradebook, you can upload CSV files containing columns of scores and then import these as assignments. To upload and import scores to the Gradebook, in the relevant course site, click Gradebook from the toolbar, and then follow the appropriate steps below.


Upload Course Grades

Note: This section assumes that you have a .csv formatted file of scores stored on your local computer. The first column of your .csv file must contain individual's usernames (e.g. student id numbers) The first row of your .csv must contain headings for the columns.

At the top of the Gradebook, click Upload/Import. Under the list of assignments, click the Upload spreadsheet (csv format) to loading dock link. You will then see a file selection form for uploading a .csv file to the Gradebook loading dock. For the Title, you must enter a unique name, which will be the name for the uploaded data file once it is in the loading dock. For the Choose a File option, you may either type in a filename or use the Browse button to browse for and select the file you want to upload. [If you browse and select the file, the full filename will appear in the Title box automatically.] Then click the save button, and upload of the file begins. The Verify Upload screen appears with a preview display of the file contents, allowing you to confirm that correct file was chosen. If you are satisfied with the match between his labels and the data, click the Ok button. If you not satisfied, click the Back button.

Once a file is uploaded to the loading dock, its score contents become available to you for importing as new Gradebook assignments, using the Import link. Uploaded files can also be deleted from the loading dock, using the Delete link.

To import an uploaded file's score contents as a new Gradebook assignment from the loading dock, click the Import link for that file. You will see the Import Preview screen showing the file contents with radio button selectors next to the title of score column. Use the radio button to select the score column to import. [Score columns can only be imported one at a time.] Click the Import Selected button.

On the Import Assignment and Scores screen, fill in the following information:

  • Title: A title is required for all Gradebook assignments. The title can be up to 255 characters in length.

  • Assignment Point Value: You must assign a point value to all Gradebook assignments. Enter a value greater than zero. Decimals are permitted, but values are limited to two decimal places.

  • Due Date: You can optionally designate a due date for a Gradebook assignment. You can manually enter the due date (e.g., 09/28/05), or you can click the calendar icon to select a date from a pop-up calendar.

Additional Options:

  • Place a checkmark in the first checkbox provided if you wish to include this assignment when calculating students' final grades. This can be changed later.

  • Place a checkmark in the second check box provided if you wish to release this assignment (i.e., make this assignment's scores available in the display to students). This can be changed later.

  • Clicking Submit will create the assignment; you will be returned to the Import Preview screen where you will see a message saying the assignment has been added. Alternatively, clicking Cancel will discard the assignment, and you will be returned to the Upload/Import page.
